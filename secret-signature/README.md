## Customization

### Adding Images
To add your uploaded images:
1. Create an `images` folder in the `secret-signature` directory
2. Add image files (e.g., `hero.jpg`, `dish1.jpg`)
3. Update background images in `styles.css` (search for `background-image`)
4. Add `<img>` tags in HTML for gallery sections

### Changing Colors
The primary accent color is `#c9a961` (gold). To change it:
1. Open `styles.css`
2. Find all instances of `#c9a961`
3. Replace with your desired color code

### Updating Contact Info
- Edit the phone number in links: `+2348136354459`
- Update address: `109 Convent Street, Agbor, Delta State`
- Modify WhatsApp number in `contact.js` line 12
